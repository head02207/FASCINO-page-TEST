# FASCINO 网页

这是 FASCINO 品牌的首页网站。

## 项目结构

```
FASCINO-page-TEST/
├── index.html      # 主页面
├── styles.css      # 样式文件
├── script.js       # JavaScript 交互脚本
└── README.md       # 项目说明
```

## 功能特点

- ✨ 现代化设计，响应式布局
- 🎨 漂亮的渐变色和动画效果
- 📱 手机和桌面完全适配
- 🔗 平滑的导航和页面滚动
- 📧 联系表单功能

## 如何使用

1. **克隆或下载项目**
   ```bash
   git clone https://github.com/head02207/FASCINO-page-TEST.git
   ```

2. **打开网页**
   - 直接双击 `index.html` 文件在浏览器中打开
   - 或使用本地服务器（推荐）：
   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 或使用 Node.js
   npx http-server
   ```

3. **自定义内容**
   - 修改 `index.html` 中的文本和链接
   - 更改 `styles.css` 中的颜色和字体
   - 替换占位符图片为你的真实图片

## 自定义步骤

### 1. 替换 LOGO 和图片
- 将你的 LOGO 和图片文件放入项目文件夹
- 在 `index.html` 中更新图片路径：
  ```html
  <img src="your-image.jpg" alt="描述">
  ```

### 2. 修改文本内容
- 编辑 `index.html` 中的所有文本
- 更新标题、描述和按钮文本

### 3. 调整样式
- 在 `styles.css` 中修改颜色（搜索 `#667eea` 和 `#764ba2`）
- 调整字体大小、间距等

### 4. 添加你的内容
- 替换占位符图片：`https://via.placeholder.com/...`
- 更新联系方式信息
- 添加社交媒体链接

## 部署建议

### GitHub Pages（免费）
1. 在仓库设置中启用 GitHub Pages
2. 选择 `main` 分支作为源
3. 网站将自动发布到 `https://head02207.github.io/FASCINO-page-TEST`

### 其他部署选项
- Vercel
- Netlify
- 自己的服务器

## 技术栈

- HTML5
- CSS3（含 Grid、Flexbox、动画）
- Vanilla JavaScript
